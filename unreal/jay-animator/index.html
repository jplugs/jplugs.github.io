Overview

JayAnimator streamlines real-time animation workflows in Unreal Engine 5

It provides smooth tween animations for Actors and Components using C++ and Blueprint and supports location, rotation, scale, and material parameter tweens, allowing you to create polished animations with minimal effort.


ðŸ“Œ Features

  • Tween Location, Rotation, Scale (Vector/Transform)
  • Tween Material Parameters (Float, Vector, Color)
  • Easy-to-use Blueprint Library and C++ API
  • Supports easing functions (Linear, EaseIn, EaseOut, etc.)
  • Configurable options for delay, looping, yoyo, and more
  • Optional completion delegate (OnComplete)
  • Works with any AActor or USceneComponent

ðŸš€ Quick Start (Blueprint)

Example: Tween Actor Location

  1. Add the TweenLocation node from jTween category in Blueprint.
  2. Set:
    • Actor: The target actor
    • From: Starting location
    • To: Target location
    • Duration: Time in seconds
    • Easing: e.g., Linear, EaseInOut
    • (Optional) OnComplete: Event to trigger after tween finishes

Example: Tween Material Parameter

  1. Use TweenMatParamFloat1 (for scalar parameters) or other material tween nodes.
  2. Provide:
    • Actor
    • From / To values
    • Duration
    • Material Options (slot name, parameter name)
    • OnComplete if needed

ðŸ’» Quick Start (C++)

Include the header in your class:

#include "jTweenBPLibrary.h"

Example: Tween Actor Location

FJTweenOptions Options;
Options.Duration = 1.5f;

UjTweenBPLibrary::TweenLocation(
    MyActor, 
    EJTweenEasing::Linear, 
    FVector(0,0,0), 
    FVector(300,0,0), 
    1.5f, 
    Options, 
    FJTweenCompleteDelegate()
);

Example: Tween Material Parameter

FJTweenMaterialOptions MatOptions;
MatOptions.ShaderParamName = "MyParam";

UjTweenBPLibrary::TweenMatParamFloat1(
    MyActor,
    EJTweenEasing::EaseInOut,
    0.f,
    1.f,
    2.f,
    MatOptions,
    FJTweenOptions(),
    FJTweenCompleteDelegate()
);

FJTweenOptions allows: - Delay: Start delay before tween begins - Loop Count: Number of times to repeat - Yoyo: Reverse back after finishing - TimeScale: Playback speed
