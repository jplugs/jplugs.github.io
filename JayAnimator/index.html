Overview

JayAnimator facilitates working with real-time animation in Unreal Engine 5.

Installation

Drop the ToolA folder into your project's Plugins directory.

Usage

Launch Unreal Engine and enable ToolA via Edit > Plugins.

Examples

// ToolA usage example
FAssetToolsModule& AssetToolsModule = FModuleManager::LoadModuleChecked("AssetTools");
AssetToolsModule.Get().CreateUniqueAssetName(TEXT("/Game/MyAsset"), TEXT(""), OutPackageName, OutAssetName);
