Overview

ToolB enhances the material editor in Unreal Engine with dynamic shader graphs.

Installation

Copy ToolB to your plugin folder and rebuild the project.

Workflow

After installing ToolB, navigate to the material editor to access new nodes.

C++ API Reference

// ToolB API usage
